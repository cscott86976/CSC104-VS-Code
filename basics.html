HTML Basics

HTML stands for hyperText markup language. HTML provides the structure of the webpage by marking up the text with various tags to organize the webpage content. HTML does not have many rules but does have some best practices, such as using 1 h1 tag on a page

History of HTML

  • HTML 1.0 developed in early 1990s
  • HTML 2.0 released in 1995
  • HTML3.2 released in 1997
  • HTML 4.01 released in 1999
  • HTML 5 released in 2014 (current/living version)

Cascading Style Sheet (CSS)

CSS is a language that is used to make HTML tags look better. It can control layouts, colors, font- basically any visual elements on the page can be changed from its default look using CSS.

History of CSS

  • CSS1 released in 1996
  • CSS2 released in 1998
  • CSS3 released in 2011- current/ living version of CSS

Basic CSS structure

                selector{
                    property: value;
                }
  • CSS selectors can be HTML element, id of an HTML element, custom class
  • CSS helps with seperation of content and looks
  • When set up globally, CSS is easy to maintain
  • Accessibility is easier to manage with CSS

Ways to include CSS on our pages

  • External - This is best practices - we have seperate CSS file
  • Embedded - this CSS applies to just 1 page - located in the head section
  • Inline - this is where we set CSS on individual HTML elements - use sparingly
